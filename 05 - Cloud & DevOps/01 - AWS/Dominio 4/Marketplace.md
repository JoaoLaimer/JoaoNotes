[[Cobrança, Preços e Suporte]]
AWS Marketplace é uma plataforma digital que permite que organizações descubram, comprem, implantem e gerenciem software, serviços e soluções de terceiros diretamente integrados à AWS. Ele facilita o acesso a milhares de produtos para diferentes casos de uso, como segurança, análise de dados, aprendizado de máquina e DevOps.
- **Soluções Pré-Configuradas**: Oferece aplicativos otimizados para rodar diretamente em ambientes AWS, como AMIs (Amazon Machine Images), contêineres e SaaS.
- **Modelos de Licenciamento Flexíveis**: Suporte para pagamento por uso, assinaturas mensais/anuais e BYOL (Bring Your Own License).
- **Categorias Diversas**: Disponibilidade de software em áreas como segurança, IA/ML, armazenamento, análise de dados, IoT, DevOps e muito mais.
- **Integração com AWS Billing**: Consolida custos de software de terceiros na mesma fatura AWS, facilitando o gerenciamento financeiro.
- **Implantação Direta**: Suporte para implantar software diretamente em serviços AWS como EC2, ECS e EKS, além de integrações com serviços SaaS.
- **Governança e Controle de Acesso**: Compatível com AWS Organizations para gerenciar permissões e implementar políticas em compras de software em ambientes multi-conta.
- **Soluções Personalizáveis**: Permite que vendedores disponibilizem software com opções configuráveis, como tamanhos de instância ou regiões específicas.
- **AWS Private Marketplace**: Habilita organizações a criarem uma versão personalizada do Marketplace para listar apenas software aprovado internamente.
- **Suporte para Contratos Privados**: Oferece a possibilidade de negociar termos personalizados diretamente com fornecedores.
**Modelo de Preços**:

- **Pague pelo uso**:
    - Cobrança com base no modelo definido pelo fornecedor (ex.: por hora, por licença ou taxa fixa).
    - Custos associados a recursos subjacentes, como EC2 ou S3, são cobrados separadamente.

**Casos de Uso Comuns no Exame AWS Cloud Practitioner**:

- Identificar o AWS Marketplace como uma plataforma para adquirir e gerenciar software de terceiros.
- Reconhecer sua integração com AWS Billing para simplificar a gestão de custos.
- Compreender a funcionalidade de Private Marketplace para gerenciar soluções aprovadas internamente.
