Standard security configuration applied to guarantee minimum security for a system, network, or application.
1. Establish a secure baseline.
2. Deploy secure baseline. across all of the organization's digital assets.
	Admins can employ GPOs for a domain-wide secure baseline, governing password policies, user rights, an audit settings.
	- **AWS Config**
3. Maintain the secure baseline on all assets.
## Master Images
1. Starts with a blank source system, install and configure any desired applications, and modify security settings
2.  Capture the image, which becomes the master image.
3. Deploy image to multiple systems.
> A master images provides secure starting points for systems. Administrators sometimes create them with templates or with other tools to create secure baselines.
> They then use integrity measurements to discover when a system deviates from the baseline.
