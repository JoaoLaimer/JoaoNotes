## Session Management 
A fundamental security component that enables web application to identify a user.

Information can be stored in database or in cookies.

Cookies must be protected because they contain client information that i being transmitted across the internet.
## Session Hijacking
A type of spoofing attack where the host is disconnected and replaced by the attacker
### Session Prediction
An attacker attempts to predict the session token to hijack that session
## Cookie Poisoning
Modifying the contents of a cookie to be sent to a client's browser and exploit the vulnerabilities in an application.
