Type of network-based attack that involves maliciously repeating or delaying valid data transmission.
## Session Hijack
Attacker alters real-time data transmission
## Replay Attack
Attacker intercepts data and decides whether to retransmit it later.
## Session Tokens
Unique data pieces that prevent session replay by attackers

> To reduce the risk of attacks, it's advisable to use the latest security protocols like WPA3 to prevent replay attacks.
