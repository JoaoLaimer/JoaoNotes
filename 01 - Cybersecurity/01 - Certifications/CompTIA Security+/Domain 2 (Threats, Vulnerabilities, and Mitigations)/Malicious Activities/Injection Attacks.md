## Lightweight Directory Access Protocol (LDAP)
A protocol for access and maintenance of distributed directory information services.
### LDAP Injection
An attack in which LDAP statement, typically created by user input, are fabricated.
- Input Validation
- Input Sanitization
## Command Injection
A threat actor is able to execute arbitrary shell commands via a vulnerable web application.
## Process Injection
A method of executing arbitrary code in the address space of a separate live process.
- Injection through DLLs
- Thread execution hijacking
- Process hollowing
- Process doppelganging.
- Asynchronous procedure calls
- Portable execution injection.
## Mitigation
- Endpoint Security Solutions
- Security Kernel Module
- Practice of Least Privilege
