There are three types of roles in IAM, basic roles, predefined roles, and custom roles.

## Basic Roles
Basic roles are the original roles that were available in the Google Cloud console, but they are broad.
**Owner**
- Invite members
- Delete members
- Delete projects
- And more.
**Editor**
- Deploy application
- Modify code
- Configure services
- And more.
**Viewer**
- Read-only access
**Billing Administrator**
- Manage billing
- Add and remove administrators

## Predefined Roles
IAM predefined roles apply to a particular Google Cloud service in a project.
IAM predefined roles offer more fine-grained permission on particular services. The permissions themselves are classes and methods in the APIs.

# Custom Roles
IAM Custom roles let you define a precise set of permissions.
